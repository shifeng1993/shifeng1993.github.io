tensorflow
tensorflow

参考文章：

  • 如何为 MacOS 构建 TensorFlow-GPU 1.8？

环境要求

  • MAC OSX 10.6+
  • 在mac上禁用SIP
  • NVIDIA Web-Drivers 驱动
  • CUDA-Drivers 驱动
  • CUDA 9.1 Toolkit 开发工具
  • cuDNN 7.0.5 神经计算加速工具
  • Python 3.6
  • xcode 8.2
  • bazel 0.10

Homebrew

如果尚未安装，Homebrew还将安装最新的Apple Command-Line-Tools
