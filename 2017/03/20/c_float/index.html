【C学习随笔】C语言的实型数据

2017-03-20
C

注：浮点运算是有误差的，用于精密计算建议用整型数据类型存储和运算。如：银行存款。

float

  • 单精度类型
    占用4个字节，32位。
    用%f进行输出

double

  • 单精度类型
    占用8个字节，64位。

用%f进行输出

long double

  • 长双精度
    占16个字节，128位。
    用%Lf进行输出
