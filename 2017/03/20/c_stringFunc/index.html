strlen()

strcat()

strlen()函数用于拼接字符串。

以下为示例，输出为：str = woshiyigehaoren

strncat()

strncat()函数用于拼接字符串。
参数：（主字符串，附加字符串，长度），
以下为示例，输出为：str = woshiyigehaoren；

strcmp()

strcmp()函数用于对比字符串。
参数：（str1，str2);

如果str1 > str2,则返回正数，
如果str1 < str2,则返回负数，
如果str1 = str2,则返回0

以下为示例，输出为：3；
